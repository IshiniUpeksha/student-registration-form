# Student Registration Form

## Project Overview
This project is part of my coursework and demonstrates a **student registration form** built with **JavaScript/jQuery validation**.
Note: The UI is intentionally simple because the focus of this task was on validation logic.

## Technologies Used
- HTML
- CSS3 (basic styling)
- JavaScript (ES6)
- jQuery (for event handling and validation)

## Features
- **Full Name**: Minimum 3 characters, letters and spaces only.
- **Email Address**: Valid email format (e.g., `name@example.com`).
- **Password**: At least 8 characters, including uppercase, lowercase, and a number.
- **Confirm Password**: Must match the password field.
- **NIC**: Supports both old (9 digits + V/v/X/x) and new (12 digits) formats.
- **Phone Number**: 10 digits, starting with 0 (e.g., `0712345678`).
- **Date of Birth**: Must be selected and user must be at least 18 years old.
- **Gender**: Required selection.
- **Agreement Checkbox**: Must be checked to proceed.
- **Inline Feedback**: Real-time error messages with red/green border cues.
- **Success Message**: Displays "Registered Successfully!" when all validations pass.

## How to Run
1. Clone this repository:
  ```bash
  git clone https://github.com/IshiniUpeksha/student-registration-form.git 
  ```

2. Open index.html in your browser.
3. Fill out the form to test the validation.
